Tips for Creating a Strong Password

Your password is strong if it meets all the following criteria:

  • Include a lowercase letter (e.g., a, b, c)
  • Include an uppercase letter (e.g., A, B, C)
  • Include a special character (e.g., !, @, #, $)
  • Be at least 8 characters long
  • contain at least one digit (e.g., 0, 1, 2)
